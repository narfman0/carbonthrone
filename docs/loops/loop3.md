# Loop 3 — Fraying

## Station State

Significant damage that wasn't caused by anything in this loop — bleed from previous iterations. The anomaly readings are worse at the start than they were at the end of loop 1. Crew members are having headaches, fragmented dreams.

---

## Opening Changes

- Orin is calm in a way that feels rehearsed
- Doss is already checking the door when you wake up
- Kaleo's cut-off line is different: *"I should note that I have run this calculation before — [drops]"*

---

## What the Player Can Discover

- Orin's personal logs (partial) if she is the companion — she controls access but is starting to slip
- Doss will reference the war obliquely if pressed; he trusts you more on his companion route
- A terminal near the player's path displays: *"I would be more useful in the field."* — Kaleo can be recruited this loop

---

## Kaleo Recruitment

The terminal message appears once. The player chooses to accept or ignore it. If accepted, Kaleo joins the party. It does not explain itself.
